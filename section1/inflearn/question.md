## 1️⃣2️⃣ 곱하기 or 더하기
> ✍️ **난이도:** 중급

여러분 위에서 어려운 개념들을 배우시느라 고생 많으셨습니다. 
이제부터는 문제를 풀어보자구요! 🎉
### 📝 문제 설명
다음과 같이 0 혹은 양의 정수로만 이루어진 배열이 있을 때, 왼쪽부터 오른쪽으로 하나씩 모든 숫자를 확인하며 숫자 사이에 '✕' 혹은 '+' 연산자를 넣어 결과적으로 가장 큰 수를 구하는 프로그램을 작성하시오. 

**주의사항:** '+' 보다 '✕'를 먼저 계산하는 일반적인 방식과는 달리, 모든 연산은 왼쪽에서 순서대로 이루어진다.

### 💻 입력 예시
```python
arr = [0, 3, 5, 6, 1, 2, 4]
```

### 🎯 예상 출력
```python
270  # 0 + 3 × 5 × 6 × 1 × 2 × 4 = 3 × 5 × 6 × 1 × 2 × 4 = 720
```

## Q2. 문자열 뒤집기
> ✍️ **난이도:** 고급 | **링크:** [원 문제 링크]
### 📝 문제 설명
0과 1로만 이루어진 문자열이 주어졌을 때, 이 문자열에 있는 모든 숫자를 전부 같게 만들려고 한다. 할 수 있는 행동은 문자열에서 연속된 하나 이상의 숫자를 잡고 모두 뒤집는 것이다. 뒤집는 것은 1을 0으로, 0을 1로 바꾸는 것을 의미한다.

### 🔍 예시 분석
예를 들어 `S = "0001100"`일 때:

1. **전체를 뒤집으면** → `"1110011"`
2. **4번째~5번째 문자를 뒤집으면** → `"1111111"` (총 2번)

하지만 처음부터 4번째~5번째 문자를 뒤집으면 한 번에 `"0000000"`이 되어서 **1번 만에** 모두 같은 숫자로 만들 수 있다.

### 💻 입력 예시
```python
s = "0001100"
```

### 🎯 예상 출력
```python
1  # 최소 횟수
```

## 🎆 보너스: 문자열 요약하기
> **난이도:** 초급 | **보너스 문제** (별도 해설 영상 없음)

### 💡 배경
강의 피드백 중, 예시 문제가 더 있었으면 좋겠다는 의견과 숙제가 너무 어렵다는 피드백이 있었습니다. 이를 반영해서 해당 문제들은 강의 자료 중간중간에 추가될 예정입니다. 🎆 아이콘이 있는 문제들은 보너스 문제입니다!
### 📝 문제 설명
입력으로 소문자의 알파벳 순으로 정렬된 문자열이 입력됩니다. 입력 문자열에 나타나는 각 알파벳의 종류와 개수를 요약하여 나타내시오.

### 📋 조건
1. 입력으로 소문자의 알파벳 순으로 정렬된 문자열이 입력됩니다
2. 각 알파벳은 중복이 가능합니다
3. 중간에 없는 알파벳이 있을 수도 있습니다

### 💻 입출력 예시

#### 예시 1
```python
# 입력
text = "abc"
# 출력
"a1/b1/c1"
```

#### 예시 2-1
```python
# 입력
text = "aaabbbc"
# 출력
"a3/b3/c1"
```

#### 예시 2-2
```python
# 입력
text = "abbbc"
# 출력
"a1/b3/c1"
```

#### 예시 3-1
```python
# 입력
text = "ahhhhz"
# 출력
"a1/h4/z1"
```

#### 예시 3-2
```python
# 입력
text = "acccdeee"
# 출력
"a1/c3/d1/e3"
```
