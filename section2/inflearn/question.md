### 1) 링크드 리스트 append 함수 만들기
**문제**  
링크드 리스트에 새 노드를 끝에 추가하는 함수를 구현하시오.

**입력**
```java
String[] trainCompartments = {"기관실", "시멘트", "자갈", "밀가루", "우편"};
```

**출력**
```
기관실 -> 시멘트 -> 자갈 -> 밀가루 -> 우편
```

### 6) 두 링크드 리스트의 합 계산
**문제**

**입력**
```java
int[] list1 = {6, 7, 8};
int[] list2 = {3, 5, 4};
```

**출력**
```
1032
```

### 11) 배달 가능 여부 판단

### 12) 더하거나 빼거나 (타겟 넘버 문제)
**문제**

**입력**
```java
int[] numbers = {1, 1, 1, 1, 1};
```
